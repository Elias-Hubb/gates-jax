# JAX Quantum Circuit Simulator — Bloch Vector Representation

A lightweight quantum circuit simulator built on [JAX](https://github.com/google/jax), operating entirely in the **Pauli/Bloch vector (coherence vector) picture** rather than the standard statevector or density matrix picture. This makes it particularly well-suited for simulating **dissipative (open) quantum circuits** and computing their steady states efficiently.

This library was developed for and used in the paper:

> **[Scaling Quantum Algorithms via Dissipation: Avoiding Barren Plateaus](https://arxiv.org/abs/2507.02043)**

The Bloch vector formalism is described in detail in:

> **[Efficient quantum circuit simulation using a multi-qubit Bloch vector representation of density matrices](https://doi.org/10.1103/PhysRevA.105.022409)**

---

## Overview

Any *n*-qubit density matrix ρ can be written as:

$$\rho = \frac{1}{\sqrt{d}} \left( F_0 + \sum_{j=1}^{d^2-1} v_j F_j \right)$$

where `{F_j}` is the normalized Pauli basis and **v** is the coherence (Bloch) vector. In this representation, every quantum gate and noise channel becomes a **real matrix** acting linearly on **v**. This library implements those matrices for a comprehensive gate set using JAX, enabling JIT compilation and automatic differentiation.

## Quickstart

```python
from gates_jax import Gate, density_matrix_to_pauli_vector, pauli_vector_to_density_matrix
import jax.numpy as jnp

# Convert a density matrix to a Pauli/Bloch vector
dm = jnp.array([[1, 0], [0, 0]], dtype=complex)  # |0><0|
v = density_matrix_to_pauli_vector(dm)

# Apply a Hadamard gate
H = Gate('H', N_qubits=1, Gate_loc=0)
v_out = H.circuit_gate() @ v
```

## Supported Gates

**Single-qubit (non-parameterized):** `H` / `Hadamard`, `X` / `PauliX`, `Y` / `PauliY`, `Z` / `PauliZ`, `S`, `T`

**Single-qubit (parameterized):** `RX(φ)`, `RY(φ)`, `RZ(φ)`, `RY_Z(φ)`

**Two-qubit (non-parameterized):** `CNOT`, `SWAP`

**Two-qubit (parameterized):** `RXX(t)` / `IsingXX`, `RYY(t)` / `IsingYY`, `RZZ(t)` / `IsingZZ`, `XXPlusYY(t)`, `XXMinusYY(t)`, `CRX(φ)`, `CRY(φ)`, `CRZ(φ)`

**Noise channels:** `AmplitudeDamping(γ)`, `Depolarizing(p)`

## Key Features

### Gate objects

```python
# Single-qubit gate on qubit 1 of a 3-qubit system
rx = Gate('RX', params=[0.5], N_qubits=3, Gate_loc=1)
M = rx.circuit_gate()   # full 4^3 × 4^3 matrix

# Two-qubit gate between qubits 0 and 2
cnot = Gate('CNOT', N_qubits=3, Gate_loc=[0, 2])
```

### Broadcast a gate to multiple qubits at once

```python
# Apply X gate to qubits 0, 2, and 4 simultaneously
x_broadcast = Gate('X', N_qubits=5, Gate_loc=[0, 2, 4])
```

### Build and run a full circuit

```python
from gates_jax import circuit_matrix

gate_list = [
    Gate('H',  N_qubits=2, Gate_loc=0),
    Gate('CNOT', N_qubits=2, Gate_loc=[0, 1]),
    Gate('Depolarizing', N_qubits=2, Gate_loc=0),
]
params = [0.05]  # depolarizing probability

U = circuit_matrix(gate_list, params)  # full circuit as a single matrix
```

### Compute the steady state of a dissipative circuit

```python
from gates_jax import calculate_steadystate_solve

# Returns the coherence vector of the steady state ρ* = C(ρ*)
v_ss = calculate_steadystate_solve(gate_list, params)
```

### Convert between representations

```python
from gates_jax import density_matrix_to_pauli_vector, pauli_vector_to_density_matrix, string_to_vector

# Density matrix ↔ Pauli/Bloch vector
v = density_matrix_to_pauli_vector(rho)
rho = pauli_vector_to_density_matrix(v)

# Get the Bloch vector for a specific Pauli string
v_ZX = string_to_vector("ZX")
```
